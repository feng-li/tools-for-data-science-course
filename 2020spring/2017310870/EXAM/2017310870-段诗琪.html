每周定时推送理想国推荐图书

本文旨在通过爬虫爬取理想国网站推荐书目，并整合推荐内容，最终实现自动登录发送邮件。

1. 爬取推荐名录

1.1 登陆网址

本部分从理想国网站（http://www.ilixiangguo.com) 上爬取首页推荐书籍名称及简介，并整理成txt格式文件。由于首页内存较大且更新缓慢，可以采取半年一次或一季一次爬取，无需设置自动爬取时间。

import re
from bs4 import BeautifulSoup
import urllib
page = urllib.request.urlopen('http://www.ilixiangguo.com')
htmlcode = page.read() 
htmlcode = htmlcode.decode('utf-8')
pageFile = open('pageCode.txt','w') # 以写的方式打开pageCode.txt
pageFile.write(htmlcode) # 写入
pageFile.close()
with open('PageCode.txt') as f:
    html = f.read()
bs  = BeautifulSoup(html, 'html.parser')
print(bs.title)

4. 不足与优化

  1. 由于理想国首页推荐书目高达167本，每周选取一本可以实现年内无需更换书籍，因此在爬取书籍方面可能存在无法更新情况。
  2. 为实现定时发送邮件功能，将上述程序封装后，使用schedule库可实现定时发送邮件。
